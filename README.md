# CSE 15 - Academic Experiments Lab Manual

Welcome to the official repository for the **CSE 15** course lab work. This repository contains a structured compilation of **Experiments 1 through 12**, covering core concepts in web development (Frontend & Full-Stack) and Object-Oriented Programming.

---

## 🛠️ Tech Stack & Skills Covered

* **Frontend Web Dev:** HTML5, CSS3, JavaScript (ES6+)
* **Modern Web Frameworks:** React.js (Components, State, Props)
* **Core Backend & OOP:** Java (Data Structures, OOPs Principles)

---

## 📋 Table of Experiments

### 🌐 Web Development (HTML, CSS, JavaScript)
* **Experiment 1: Basic Structure & Semantic HTML**
    * *Objective:* Designing a well-structured web page using structural and semantic tags.
* **Experiment 2: Advanced Styling with CSS Grid & Flexbox**
    * *Objective:* Building responsive layouts, styling image cards, and creating interactive UI elements.
* **Experiment 3: Client-Side Validation using JavaScript**
    * *Objective:* Using DOM manipulation and regular expressions to validate user input forms.
* **Experiment 4: Asynchronous JavaScript (Promises & Fetch API)**
    * *Objective:* Fetching dynamic data from public REST APIs and rendering it into UI components.

### ⚛️ Modern Frontend (React.js)
* **Experiment 5: Introduction to React Components**
    * *Objective:* Setting up a React application and creating reusable functional/class components.
* **Experiment 6: State & Props Management**
    * *Objective:* Handling dynamic user interactions, managing local component state (`useState`), and passing props.
* **Experiment 7: React Hooks & Lifecycle**
    * *Objective:* Working with `useEffect` for data fetching, synchronization, and handling side-effects.
* **Experiment 8: React Form Handling & Conditional Rendering**
    * *Objective:* Managing controlled components, form submissions, and rendering UI conditionally.

### ☕ Object-Oriented Programming (Java)
* **Experiment 9: Basic Class, Objects & Constructors**
    * *Objective:* Implementing the blueprint of a real-world entity using custom classes, methods, and parameterized constructors.
* **Experiment 10: Pillars of OOPs (Inheritance & Polymorphism)**
    * *Objective:* Demonstrating method overriding, method overloading, and reusing code using single/multilevel inheritance.
* **Experiment 11: Interface & Abstract Classes**
    * *Objective:* Achieving 100% abstraction in Java and handling multiple inheritance using Interfaces.
* **Experiment 12: Exception Handling & File I/O**
    * *Objective:* Gracefully catching runtime errors using `try-catch-finally` blocks and performing basic file reading/writing operations.
